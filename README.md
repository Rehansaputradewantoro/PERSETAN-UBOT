### Tutorial Deploy To VPS

-  ( `git clone https://github.com/ramadhani892/RAM-UBOT` )
-  ( `cd RAM-UBOT` )
-  ( `bash setup` )
-  isi vars yang diminta, kalo ga diminta ga usah diisi
-  Buka SCRREN di VPS Anda ( `screen -S RAM-UBOT` )
